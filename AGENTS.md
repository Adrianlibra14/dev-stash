# Backend

## Commands

- **Dev server**: `npm run dev` start dev server with nodemon + ts-node (hot reload)

# Frontend

## Commands

- **Dev server**: `npm run start` start Angular dev server with `ng serve` (hot reload)
- **Build**: `npm run build` production build with `ng build`
- **Watch**: `npm run watch` dev build with watch mode and development configuration
- **Test**: `npm run test` run unit tests with `ng test`
